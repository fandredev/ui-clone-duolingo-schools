## Clone this repo

```
git clone git@github.com:fandredev/ui-clone-duolingo-schools.git
```

## Go to directory

```
 cd ui-clone-duolingo-schools
```

## Install dependencies

```
 yarn install
```

## Run server

```
 yarn run dev
```
